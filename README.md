# relati-py

Just run command:

```bash
python3 main.py
```

And relati game will be started.

```
|   | A | B | C | D | E | F | G | H | I |
| 1 |   |   |   |   |   |   |   |   |   |
| 2 |   |   |   |   |   |   |   |   |   |
| 3 |   |   |   |   |   |   |   |   |   |
| 4 |   |   |   |   |   |   |   |   |   |
| 5 |   |   |   |   |   |   |   |   |   |
| 6 |   |   |   |   |   |   |   |   |   |
| 7 |   |   |   |   |   |   |   |   |   |
| 8 |   |   |   |   |   |   |   |   |   |
| 9 |   |   |   |   |   |   |   |   |   |
turn to O: 
```

Enter the coordinates (e.g. `E5`) to place the symbol on the board for play game.
